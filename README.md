# Capacitación para Evangelistas - Fundamentos del Evangelismo 🔥

Este proyecto es un sitio web estático diseñado como recurso educativo y práctico para equipar a evangelistas. Contiene guías visuales, analogías detalladas y dinámicas interactivas para explicar el Evangelio de manera clara y efectiva.

## 📋 Contenido del Curso

El material cubre los 4 fundamentos clave del evangelismo bíblico:
1.  **El Amor de Dios**: Entendiendo la motivación divina.
2.  **La Condición Humana**: El problema del pecado.
3.  **Jesucristo**: La única solución y puente.
4.  **La Respuesta**: Arrepentimiento, fe y bautismo.

## 📂 Estructura del Proyecto

El sitio consta de las siguientes páginas:

*   **`index .html`**: Página principal que sirve como índice del curso. Incluye versículos clave, ejemplos prácticos y ventanas modales con dinámicas grupales.
*   **`Analogia virus.html`**: Explicación detallada de la analogía del "Virus en el celular" para ilustrar el pecado y la necesidad de un "reset" divino.
*   **`Analogia puente.html`**: Analogía clásica del abismo entre el hombre y Dios, y cómo Jesús es el único puente.
*   **`Analogia incendio.html`**: Ilustración poderosa sobre el sacrificio del Padre y el Hijo (el bombero) para explicar el costo del amor de Dios.
*   **`Objeciones.html`**: Guía práctica con respuestas bíblicas y lógicas a las objeciones más comunes durante el evangelismo (ateísmo, relativismo, "soy buena persona", etc.).

## 🚀 Cómo usar este repositorio

1.  **Clonar o Descargar**: Descarga los archivos a tu computadora.
2.  **Abrir**: Haz doble clic en el archivo `index .html` para abrirlo en tu navegador web favorito (Chrome, Edge, Firefox, etc.).
3.  **Navegación**:
    *   Usa el menú superior para ir a cada sección.
    *   Haz clic en las tarjetas de "Dinámicas" para ver instrucciones paso a paso en ventanas emergentes.
    *   Usa los enlaces "Ver más" para profundizar en las analogías específicas.

## 🛠 Tecnologías

*   **HTML5**: Estructura semántica.
*   **CSS3**: Diseño responsivo, animaciones suaves (scroll reveal), efectos de vidrio (glassmorphism) y transiciones.
*   **JavaScript (Vanilla)**: Lógica para menús de navegación, ventanas modales y animaciones al hacer scroll.

## ✨ Características Destacadas

*   **Diseño Responsivo**: Funciona bien en móviles y escritorio.
*   **Interactivo**: Ventanas modales para mantener al usuario en contexto.
*   **Visual**: Uso de emojis, colores temáticos y metáforas visuales.
*   **Bíblico**: Todas las enseñanzas están respaldadas con referencias bíblicas directas.
